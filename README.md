# DIY-UCY

## DIY E-Juice calculator
A simple terminal based E-Juice mixing calculator.

### Prerequisites
- ncurses
- meson

### Build Instructions
```sh
git clone https://github.com/Rewindz/diy-ucy.git
cd diy-ucy
meson setup build
cd build
meson compile
```
